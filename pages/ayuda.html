SECCIÓN DE AYUDA

En esta sección de ayuda verás una pequeña lista de los primeros pasos a realizar para hacer una página web de manera correcta, utiliziando paso a paso los conceptos aprendidos en coder house, codificando de manera correcta, eficiente y siguiendo todos los pasos. Todo esto aplicable ya sea para HTML, CSS, Bootstrap e incluso las librerias de animaciones como animate o wow.js.

  • Hacer diseño en figma o adobe xd.
  • Maquetar todo en HTML, utilizando etiquetas semánticas y divs para tener contenedores.
  • Aplicar flexbox y/o grids para estructurar la página y acomodar a los elementos.
  • Comenzar a aplicar CSS, trabajando con clases preferiblemente utilizando metodo bem.
  • Ser organizado con las medias queries y tener cuidado con los corchetes.
  • Aplicar Bootstrap en alguna sección si es necesario, por ejemplo en la Nav, para darle funcionamiento en mobile.
